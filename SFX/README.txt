Dépose ici tes effets sonores (.wav ou .mp3).

Nomme-les par catégorie (le mot doit être dans le nom du fichier) :
  - impact_xxx   -> joué au tout début (accroche du hook)
  - whoosh_xxx   -> joué à chaque changement de clip
  - riser_xxx    -> réservé pour plus tard

Exemples : impact_boom.wav, whoosh_fast.mp3, impact_hit.wav

L'app pioche automatiquement dedans quand ✨ Boost Hook est activé.
Dossier vide = boost visuel seulement (zoom + cuts rapides + flash), sans son ajouté.
