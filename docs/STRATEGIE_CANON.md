# FLOWERS CHROME — CANON V1

> **Statut : Canon V1.** Vrai **jusqu'à preuve du contraire.** Toute expérimentation future peut le modifier si la donnée le justifie. Ce ne sont pas des vérités absolues — ce sont des **décisions stratégiques assumées** sur la base de notre compréhension actuelle (≥80 %). On ne rouvre plus le débat sans donnée réelle qui le contredit.

## Les décisions stratégiques V1
1. **Marché du SENS, pas du produit.** Fonction morte (smartphone) → valeur d'une montre non-connectée ~100 % symbolique. *(compréhension marché, solide)*
2. **Décision V1 — on joue le jeu du GOÛT accessible** (pas celui de l'argent/Rolex). Hypothèse : c'est le terrain où un primo-acheteur à ~180 € peut nous croire légitimes.
3. **Décision V1 — cible = le primo-acheteur aspirant du SWEET SPOT** : en sait assez pour vouloir une vraie (automatique, saphir), pas assez pour bricoler / snober les mods.
4. **Décision V1 — le concurrent n°1 est l'inertie**, et le budget « plaisir » est disputé par tout le portefeuille (sneakers, parfum…). Pas de coût d'inaction naturel → **urgence à fabriquer** (rareté) ou **capter** (déclencheurs de vie).
5. **Le tunnel MENTAL fuit à 3 endroits** : Étape 5 (rejet du mod), Étape 6 (inertie), Étape 9 (réception). *(observation de parcours, à valider mais structurante)*
6. **Décision V1 — vendre identité + confiance ; ne pas éduquer techniquement.** Raisonnement : la valeur d'un mod s'effondre si le prospect découvre le coût DIY / le discours puriste → éduquer = le rapprocher du seuil de rejet.
7. **Décision V1 sur le mot « mod » :** nous choisissons **volontairement de ne pas mettre « mod » au centre de la communication**, car notre hypothèse est qu'il réduit la valeur perçue auprès du primo-acheteur. *(Décision assumée, révisable — pas une vérité.)*

## Ce que le Canon V1 INTERDIT (garde-fou de production)
- Lexique : « mod / SKX / parts / build / DIY », prix de composants, jargon finition/revente.
- Contenu « éducatif technique » qui apprend au prospect à devenir puriste/bricoleur.
- Noms qui signalent la réplique (« Seiko Daytona / Datejust »).

## Ce que le Canon V1 EXIGE
- Axe **goût accessible** : identité, ressenti, look, appartenance, confiance.
- Objections traitées par **identité + confiance**, jamais par justification technique.
- **Rareté** (anti-inertie) + capter les **déclencheurs de vie**.
- Soigner le **moment de réception** (fierté → preuve sociale → répare aussi Étapes 4 & 6).

## Révision
Toute donnée réelle (DM, ventes, tests) qui contredit une Décision V1 déclenche sa mise à jour → Canon V2. Sinon, on exécute.

*Figé le 2026-07-01. Phase suivante = EXÉCUTION (branding, profil, générateurs, contenu).*
